A) HTML Structure - top

This theme is a fixed layout. All of the HTML code carries comments to note the start and end of a section, a comment is text held between these tags. The general template structure is the same throughout the template. Here is the general structure.

B) CSS Files and Structure - top

There are 2 CSS files in this theme.

  • workless.css - This file contains browser reset, global styles, preset classes and media queries. Location: base/css/workless.css
  • plugins.css - This file contains CSS for any plugins used, such as tooltips, sliders etc... Location: base/css/plugins.css

All the stylesheets are separated into sections using comments like below:

	/* Header Section ------------------------*/

	some code

	/* Main Section ------------------------*/
	
	some code
	
	/* Sidebar Section ------------------------*/
	
	some code
	
	/* Footer ------------------------*/
	
	some code
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme uses various Javascript functions which are stored in 2 files.

  1. scripts.js
  2. plugins.js
  1. scripts.js contains all site wide Javascript functions and plugins.js again contains any Javascript for plugins used.

D) Theme Installation - top

It is extremely simple to install your new theme, the details are outlined below.

Installation via FTP:

  1. Simply browse to the directory you installed PyroCMS in and then go into the addons/theme folder. Just upload your new theme_folder into here. Be sure that you unzip the folder before uploading.
  2. Login to your PyroCMS admin panel and from the top menu browse to design/themes. From here simply select your new theme and click save. From this menu you can also access any theme options such as colors and backgrounds by clicking on the 'options' button.

E) Theme Options - top

Theme Options: This theme comes with options for the Slider, Colors and Backgrounds. All settings can be accessed via the 'options' section of the admin panel. To edit or change the social links in the footer simply open base/views/partials/footer.html and follow the commented instructions. Same with the slider, simply open base/views/partials/slider.html and follow the instructions.

F) Classes and Usage - top

This theme comes with various pre-set classes that enable you to create items such as columns, lists and boxes with ease.

  1. Columns - one_full, one_half, one_third, two_third, one_quarter. Note: You can add the class "last" to any right column to remove it's right margin. Usage: 
    <div class="one_full">content goes here...</div>
  2. Notices - alert info, alert warning, alert success. Usage 
    <div class="alert success">content goes here...</div>

This is just a tiny snippet of the classes available and there are far too many to document. Simply open workless.css and you can see all the available classes.

The theme demo displays additional sections in the sidebar and footer, these are simply PyroCMS Widgets. To create these head to Content > Widgets in the admin panel, create areas 'sidebar' and 'footer' then simply drag and drop any widgets you'd like to display.

This theme uses the jQuery plugin colorbox. This plugin enables images and html to be opened inside the same page via a fancy lightbox effect. To enable simply give the anchor tag that wraps an image class="zoom" or if you wish to open another webpage give that anchor tag class="iframe".
